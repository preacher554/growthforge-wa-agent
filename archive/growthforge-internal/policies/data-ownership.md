# Data Ownership Policy

## Customer-owned

- WhatsApp number.
- Business data supplied by customer.
- Customer conversation history.
- Customer contact details.
- Product/service information.

## GrowthForge-owned

- Agent framework.
- Skills and package templates.
- Prompt architecture.
- Operating SOPs.
- Internal QA process.
- Generic reusable improvements not containing client confidential data.

## Shared operational use

GrowthForge may process customer data only to operate, maintain, and improve that customer's WA Agent service.

Tenant data must not be used across other tenants.
